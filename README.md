# 📈 Ricardo Alemão | Portfólio de Apostas Esportivas

Este é o site oficial do **Ricardo Alemão**, um portal focado em conversão para apostadores esportivos que buscam profissionalismo e resultados consistentes no futebol. O site funciona como uma Landing Page de alta performance, direcionando usuários para os grupos Free e VIP (Hubla).

![Preview do Projeto](https://img.shields.io/badge/Status-Online-brightgreen)
![Layout-Responsivo](https://img.shields.io/badge/Layout-Responsivo-blue)

## 🚀 Tecnologias Utilizadas

O projeto foi construído com foco em velocidade de carregamento e design minimalista:

* **HTML5 Semantic:** Estrutura clara para melhor indexação.
* **Tailwind CSS:** Estilização moderna e utilitária via CDN.
* **Google Fonts (Inter):** Tipografia focada em legibilidade e seriedade.
* **Mobile First:** Totalmente otimizado para celulares e tablets.

## 🛠️ Funcionalidades

- [x] **Seção Hero:** Chamada de alto impacto focada em lucro e futebol.
- [x] **Conversão Dupla:** Botões destacados para Grupo Free (Telegram) e Grupo VIP (Checkout Hubla).
- [x] **Seção Sobre:** Espaço para autoridade pessoal com métricas de assertividade e volume de membros.
- [x] **Social Link:** Integração direta com o Instagram.
- [x] **Design Dark Mode:** Estética premium que transmite exclusividade.
